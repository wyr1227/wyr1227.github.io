Hugo搭建个人博客

@Wang · Feb 1, 2018 · 1 min read

这篇文章主要描述自己是如何厉害hugo搭建在github上的个人blog的使用教程，帮助想搭建个人blog的人提供很好的入门资料,本文以windows为例子。 hugo只是众多选择之一同样的hexo也是很优秀的~~

下载hugo

Hugo 点击进行下载选择适合自己系统的版本。

本次示例下载的是 hugo_0.35_Windows-64bit.zip 压缩后只需要 hugo.exe

$ makdir blog

$ cd blog

$ makdir bin

之后把下载来的 hugo.exe 放到新建的 bin 文件夹里

设置系统环境变量

这里就不多说了流程如下

计算机 > 属性 > 高级系统设置 > 环境变量 > 系统变量里的Path > Path在上新增的 hugo.exe 别忘了分号

hugo version

查看是否配置成功

然后新增个人网站目录

$ hugo new site blog

$ cd blog

$ d themes

接下来设置主题 Hugo 的主题有很多都在这里主题

选择自己喜欢的本文选择的是 hugo-nuo

在themes文件夹里

$ git clone https://github.com/laozhu/hugo-nuo

然后进入 blog 根目录

> $ hugo server --theme=hugo

localhost:1313

观看效果

配置主题文件

发现效果不对头呀？？？？原因是你没有引用或者说配置该主题的config.toml

所以我们看看这个主题的exampleSite文件夹。 你会找到一个名为config.toml的文件。 要使用它，将

config.toml复制到Hugo站点的根文件夹中。

关于config.toml的详情请查看hugo-nuo

新建文章

在根目录下

hugo new post/first.md

一定要注意新建的md文件里

  title: "Hugo搭建个人博客"
  date: 2018-02-01T16:26:09+08:00
  draft: false
  draft 表示的是否为草稿，是草稿就不显示

接下来就是上传github了

首先在GitHub上创建一个Repository，命名为：wyr1227.github.io （wyr1227替换为你的github用户名）。

在站点根目录执行 Hugo 命令生成最终页面：

> $ hugo --theme=hugo-nuo --baseUrl="https://wyr1227.github.io"

如果一切顺利，所有静态页面都会生成到 public 目录，将pubilc目录里所有文件 push 到刚创建的 Repository的 master 分支。

$ cd public

$ git init

$ git remote add origin https://github.com/wyr1227/wyr1227.github.io.git

$ git add -A

$ git commit -m “init”

$ git push -u origin master

刚部署要等会才会有效果切记~~

然后就是访问http://coderzh.github.io/
